# Bakery Sales Tracker (Java)

##  Overview
This is a simple Java program that helps track daily sales for a bakery and calculates total and average sales. 
## Features
- Input number of days
- Enter daily sales
- Calculate total and average
- Display a clean sales report

## Tech Stack
- Language: Java
- IDE: IntelliJ IDEA

##  How to Run
1. Clone this repository
2. Open in IntelliJ IDEA
3. Run `BakerySalesTracker.java`
